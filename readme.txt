Add social button to your wordpress website for share your post to social network such Facebook,LinkedIn,Twitter and Google.

== Description ==

Social Share Plugin allows you to add google, facebook,twitter and linked in buttons in posts.
This lets you have an integrated site without a lot of coding,and still letting you customize it exactly the way you'd like.

First, activate and set up the base plugin, which makes your site haveall social functionality.
Unlike other social plugins for wordpress, this one helps you add all social buttons in posts.

Requires Wordpress 3.0 and PHP 5.

== Installation==


1. Upload `advance-social-sharer` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure from WSS in left menu.

== Frequently Asked Questions ==

= I cannot see Facebook Button icon in the post after plugin installation =

1. In WordPress admin panel go to "Plugins", find "Advance Social Sharer" and press "Activate".

= How I can deactivate the plugin=

1. In WordPress admin panel go to "Plugins", find "Advance Social Sharer" and press "Deactivate".
